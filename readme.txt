== Description ==

No Browse Happy is a simple (and useless) WordPress Plugin, that can be used to hide the Browse Happy logo which is displayed in the footer of all WordPress admin pages when viewed using IE.

I created this Plugin to [stop an argument in the WordPress hackers mailing list][1].

 [1]: http://sudarmuthu.com/blog/2008/03/15/no-browse-happy-plugin.html "No Browse Happy WordPress Plugin"
	
== Installation ==

Upload the file to your WordPress Plugin directory and then activate it from the Plugins page.
